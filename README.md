# Padrão Command - Basquete

Implementação do padrão de projeto Command aplicado ao contexto de testes em basquete.

## Como executar

### Pré-requisitos
- Java 11 ou superior
- Maven 3.6 ou superior

### Executar os testes

```bash
mvn test
```

### Compilar o projeto

```bash
mvn clean compile
```
